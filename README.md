# Even G2 Apps Collection

A collection of applications built for Even Realities G2 smart glasses.

## Apps

### [MoonWalker](./MoonWalker)

Minimalist AR navigation app that displays directions and distance on your G2 glasses. Features dual map services (Amap + Photon), navigation history, and stoic philosophical quotes.

**Key Features:**
- Minimalist AR navigation with arrow + distance display
- Dual map services: Amap (China) + Photon (Global)
- Navigation history with localStorage
- Stoic quotes rotation every 10 minutes
- Real-time updates every 2 seconds

**Tech Stack:** Even Hub SDK · Vite · Vanilla JS · Amap/Photon APIs
